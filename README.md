# ZenHub Dashboard

Panel de productividad y gestión de tareas con Pomodoro, tracker de hábitos y clima en tiempo real.

## Características
- ⏱️ Técnica Pomodoro con 25min
- 📋 Gestor de tareas dinámico
- 🎯 Rastreador de hábitos semanal
- 🌤️ Información climática en vivo
- 📊 Estadísticas semanales
- 💾 Persistencia con localStorage
